Django project與 app差異

特性 Project App
創建命令 django-admin startproject python manage.py startapp
內容 包含settings.py, urls.py等全局配置 包含models.py, views.py等應用文件
例子 整個網站 用戶認證、投票系統 (功能)
關係 一個project可以包含多個app 一個app可以屬於多個project
主要作用 管理app 實現功能
數量 一個Django實例通常只有一個project 一個project可以有多個app

總結：

  1. Project 是整個網站的設定，底下可包含App。

  2. App 專注執行功能。

  3. 一個 Project 可以包含多個 App。

假設部落格網站(一個project)，你可以依據功能拆分多個功能(app)，如發文功能、留言功能、投票功能。當然你也可以都寫在一起，只是拆分有利於維護與分工。
