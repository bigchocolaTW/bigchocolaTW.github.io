npm init 與 npm create差異

簡單來說，同義詞，等價，別名。

可參閱npm create和npm init、彻底弄懂 `npm innpm init vue@latest ， How ‘npm create vue@latest’ works

當然，官方文檔也是必需的。詳細的npm指令可以參考[指令] npm cli & package.json

npm init 當前目錄創建package.json文件，如果已有package.json文件，會詢問一些問題讓使用者修改文件資訊(其實就是初始化時會問的)。如下:

npm init -y init 的問題全部yes

npm init <安裝包名稱> 安裝指定npm包，預設會更新寫進package.json內
